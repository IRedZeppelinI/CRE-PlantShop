# PlantShop (Projeto CRE 89104)

Este é o repositório do projeto **CLOUD-89104/2025**

## Descrição do Projeto

O objetivo é desenvolver uma solução de software *cloud-native* para entusiastas de botânica.

O projeto será totalmente desenvolvido com recurso a serviços PaaS (Platform-as-a-Service) do Microsoft Azure.

## Estado Atual

**WIP (Work in Progress):**

## Stack Tecnológica (Planeada)

* **Backend & Frontend:** .NET 9 (Blazor Server)
* **Arquitetura:** Clean Architecture
* **Serviços Azure:**
    * Azure App Service (para Containers)
    * Azure SQL Database
    * Azure Cosmos DB (NoSQL)
    * Azure Blob Storage
    * Azure Service Bus   
* **DevOps:** Azure DevOps (Pipelines)
